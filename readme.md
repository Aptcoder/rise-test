### Project structure

```
-- config (project wide configurations)
-- tests (all tests are here)
-- src
---- controllers (web controllers - they decide what happens with api requests )
---- services (core logic)
---- repository (Data access layer (or sort of).
---- migrations (database migrations)
---- utils
-------- interfaces
-------- dtos

```

### Tools used

- Node.js
- Express
- PostgreSQL
- TypeORM
